## Supabase Setup

To complete Supabase setup:
1. Go to supabase.com, sign up free, create a new project
2. In your project dashboard go to Settings → API and copy your Project URL and anon public key
3. Copy .env.local.example to .env.local and paste your credentials
4. Run: npx supabase db push (this applies the schema)
5. In the Supabase dashboard, go to SQL Editor and run the contents of supabase/seed.sql

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
